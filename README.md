An **open-source, local-first personal life OS** — designed to be the operating system for your life. All data stays on your device, always.

**Core Modules:**
- 🔒 **Privacy Vault** — 100% local-first, zero cloud dependency
- 📊 **Habit Engine** — Analytics-driven habit tracking with streaks
- 💰 **Finance Hub** — Income, expenses, budgets & insights
- 📝 **Notes** — Rich markdown editor with backlinks
- 🏋️ **Workout Lab** — Plan, track & analyze fitness routines
- 🍳 **Recipe Studio** — Meal planning & nutrition tracking
- 🧠 **Mind Mapper** — Visual knowledge graphs
- ⏱️ **Time Forge** — Pomodoro & deep-work scheduler
